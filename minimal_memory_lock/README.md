# minimal_memory_lock

## Requirements 

Adjust permissions for memory locking see (https://docs.ros.org/en/foxy/Tutorials/Real-Time-Programming.html#adjust-permissions-for-memory-locking)
 or run as root

## How to run

```bash
$ ros2 run minimal_memory_lock minimal_memory_lock
```
